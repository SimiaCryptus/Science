


# Functional

* Describe to Planner
    * Text description of the task
      * Describes all critical configuration options
      * Describes purpose of the task
      * Describes expected output
    * Configuration
      * Description annotations
      * Data validation rules
* Final Output back to Planner
    * Should contain length-managed summary/results of the task
    * Should note all files created; both path and link
* Progress Monitor UI
  * Critical for some uses, e.g. chat mode
  * Irrelevant for fully automated uses
  * Use markdown transcript files for full details
  * Output code, markdown, images, etc as files, linking from the UI

# Requirements and Best Practices
